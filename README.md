# 🌱 KrishiMitra AI

AI-powered Plant Disease Detection Assistant for Farmers

## 📌 Overview

KrishiMitra AI is a deep learning-based web application that helps farmers identify crop diseases by simply uploading an image of a plant leaf. The system analyzes the image using a trained AI model and provides disease predictions, treatment recommendations, multilingual support, and voice assistance.

The project aims to make crop disease diagnosis faster, more accessible, and farmer-friendly.

---

## Features

*  Plant Disease Detection using AI
*  Image Upload & Camera Capture Support
*  Voice-based Disease Information
*  Multilingual Interface

  * English
  * Hindi
  * Punjabi
  * Tamil
*  Detailed Treatment Recommendations
* 📊 Prediction History Tracking
*  Responsive User Interface
*  Cloud Deployment on Render

---

##  AI Model

* Model: EfficientNetB0
* Framework: TensorFlow / Keras
* Dataset: PlantVillage Dataset
* Classes: 38 Plant Disease Categories
* Dataset Size: 54,000+ Images
* Validation Accuracy: 94.74%

##  Problem Statement

Farmers often struggle to identify crop diseases at an early stage due to lack of expert guidance and limited access to agricultural resources.

KrishiMitra AI provides an AI-powered solution that enables quick disease detection and treatment suggestions directly from a mobile phone or computer.
